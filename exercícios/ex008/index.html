Formatações adicionais

Marca-Texto

Podemos criar também um texto marcado ou Marca-Texto usando a tag "MARK".

Texto Grande

Podemos criar um texto grande usando a tag "big" (<== NÃO semântica).

Texto Pequeno

Podemos criar um texto pequeno usando a tag "small".

Texto-Deletado

Indica um texto que é para ler mas NÃO é para ser considerado usando a tag "del".

Texto-Inserido / Texto Sublinhado

Ele indica que o texto foi adicionado ali depois usando a tag "ins".

Este texto usa a mesma formatação porém ele NÃO é semântico pois utiliza a tag "u".

Texto-Sobrescrito

Este comando levanta e diminui o caractere, serve para colocar coisas do tipo; x20+3.

Texto-Subscrito

Este comando abaixa e diminui o caractere, serve para colocar coisas do tipo; H2O.
